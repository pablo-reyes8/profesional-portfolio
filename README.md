# 🚀 Pablo Reyes - Professional Portfolio

Welcome to my interactive portfolio. A modern and visually stunning website where I showcase my projects, skills, and expertise in **Machine Learning, Data Science, and Software Engineering**.

## ✨ Features

- **Interactive Interface**: Smooth animations and particle fields with realistic physics
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Performance Optimized**: Built with Vite and React 19 for a blazingly fast experience
- **3D Visualization**: Advanced visual effects with Three.js
- **Smooth Navigation**: Auto-scrolling between sections

## 🛠️ Tech Stack

- **Frontend Framework**: [React 19](https://react.dev)
- **Build Tool**: [Vite 7](https://vitejs.dev)
- **Language**: [TypeScript](https://www.typescriptlang.org)
- **3D Graphics**: [Three.js](https://threejs.org)
- **Styling**: CSS3 with custom animations
- **Hosting**: GitHub Pages

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/pablo-reyes8/profesional-portfolio.git
cd profesional-portfolio

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The site will be available at `http://localhost:5173`

### Production Build

```bash
# Compile and optimize
npm run build

# Preview the build
npm run preview
```

### Deploy to GitHub Pages

```bash
npm run deploy
```
